Projects

Larasketch - Esboço de aplicação web para inicialização rápida de novos projetos. Empacotado com funcionalidades comums em sistemas web, é um belo demostrativo das capacidades do Laravel 5.1.
Autenticação Delegada - Este trabalho académico demonstra o uso de código de barras bidimensionais, aplicados em um processo de autenticação, como uma maneira de evitar o roubo malicioso de credenciais de usuários.
Calculadora de Rede - Software simples, construído em C para encontrar rapidamente valores fundamentais presentes em um ambiente de rede, tais como o ip em formato binário, máscara, endereço de broadcast e assim por diante.
Python - Apresentação da linguagem Python através de uma perspectiva analítica, científica e bem resumida. Explora a história da linguagem, expõe tópicos sobre análise léxica, explica a estrutura da linguagem e mostra exemplos práticos de uso.
Sistemas Operacionais - Aprenda alguns conceitos básicos de sistemas operacionais com o Minix . Um sistema operacional de código livre e aberto, projetado para ser confiável, com um enfoque educativo e que permite o aprendizado rico de importantes conceitos de sistemas operacionais.
Gists - Ótima maneira de compartilhar trechos de código. Você pode compartilhar arquivos individuais, partes de arquivos ou aplicações completas. Confira em https://gist.github.com.
Calculadora de Juros - Verifique facilmente o valor de um financiamento de acordo com a Tabela SAC, muito utilizado em financiamentos de imóveis. Verifique o código-fonte em https://github.com/allisonmachado/interests.
